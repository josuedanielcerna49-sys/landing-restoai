# Guion corto + idea de demo de Lili — RestoAI

## Nombre del guion

**RestoAI: del cliente al panel, todo conectado**

## Duración estimada

**45 a 60 segundos**

## Objetivo

Mostrar de forma clara cómo RestoAI ayuda a un restaurante a atender clientes con Lili y organizar la operación interna desde el panel administrativo.

El guion debe comunicar que RestoAI no es solo un chatbot, sino un sistema completo para restaurantes que conecta la atención del cliente con pedidos, reservas, menú, clientes, reclamos y supervisión humana.

## Público objetivo

Dueños, administradores o encargados de restaurantes, cafés y negocios de comida que reciben pedidos, reservas o consultas por WhatsApp, redes sociales o llamadas.

# Idea de demo de Lili

## Nombre de la demo

**Cliente hace pedido y reserva con Lili**

## Objetivo de la demo

Mostrar cómo un cliente puede usar Lili para consultar el menú, hacer un pedido o reservar, y cómo esa información aparece en el panel administrativo para que el restaurante le dé seguimiento.

## Flujo de la demo

1. Abrir el chat del cliente con Lili.
2. El cliente saluda y pregunta por el menú.
3. Lili muestra opciones del menú.
4. El cliente pide ayuda para hacer un pedido o reservar una mesa.
5. Lili guía al cliente con la información necesaria.
6. El pedido o reserva queda registrado.
7. Se entra al panel administrativo.
8. El equipo revisa el pedido o reserva.
9. Se muestra que el equipo puede supervisar el chat.
10. Si hace falta, el equipo puede tomar control de la conversación.

## Conversación ejemplo para la demo

Cliente: “Hola, quiero ver el menú.”
Lili: “¡Hola! Claro, te puedo ayudar con el menú, pedidos, reservas, horarios o reclamos.”

Cliente: “Quiero hacer un pedido para recoger.”
Lili: “Perfecto. Te muestro las opciones disponibles y después confirmamos tu pedido.”

Cliente: “También quiero reservar una mesa para 4 personas.”
Lili: “Claro. Decime la fecha y hora para revisar la disponibilidad.”

## Mensaje que debe quedar claro

RestoAI permite que el restaurante atienda mejor al cliente y mantenga organizada la operación interna.

Lili ayuda con la atención inicial, pero el equipo del restaurante puede supervisar, intervenir y gestionar todo desde el panel.

## Resultado esperado de la demo

La persona que vea la demo debe entender que RestoAI sirve para:

* Atender clientes con Lili.
* Mostrar menú.
* Tomar pedidos.
* Gestionar reservas.
* Registrar reclamos.
* Supervisar conversaciones.
* Conectar la atención del cliente con el panel administrativo.
* Dar más control al equipo del restaurante.
